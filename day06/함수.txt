함수는  function, 기능 이라는 뜻을 가지고 있는데
지금은 코드를 그룹으로 묶는다고 생각해도 됨,

기본적인 문법은

function 함수이름() {
    실행될 코드
}

함수이름(); 을 실행하면
내부에 있는 코드를 가져와 실행 시킨다.

function 함수이름(매개변수) {
    실행될코드
    이 안쪽에서 매개변수를 이용할 수 있다!
}

메개변수를 여러게 사용시

function 함수이름(매개변수이름1,2,3) {
    실행될코드
    이 안쪽에서 매개변수를 여러개 사용할 수 있다!
}
